Introduction

We propose the Dialogue-Level Dependency Parsing (DiaDP) task for Chinese to promote research in dialogue understanding. This task falls within the domain of fine-grained semantic structure analysis in multi-turn dialogues. In the DiaDP task, models and systems are expected to generate accurate dependency structures that capture the semantic relationships between utterances and their elements across dialogue turns.

To comprehensively evaluate model performance, the task assesses both inner parsing, focusing on dependencies within an Elementary Discourse Unit (EDU), and inter parsing, capturing dependencies across dialogue EDUs.

Challenge Task Definition and Metrics

Participants are required to construct models that can accurately parse dialogue-level dependency structures. Given a multi-turn dialogue, models must predict dependency structures that capture both inner-turn relationships (within individual turns) and inter-turn relationships (across turns), ensuring fine-grained semantic coherence.

Evaluation Metrics:

  • Inner UAS and LAS: Measures the model’s performance within EDUs.
  • Inter UAS and LAS: Measures performance across EDUs.

Dataset

Our dataset is structured specifically for the Dialogue-Level Dependency Parsing (DiaDP) task. Each dialogue is represented in JSON format, consisting of turns, utterances, and their dependency relationships. 

{
  "id": "10374",
  "dialog": [
    { "turn": "0", "utterance": "请您稍等，正在为您确认此前咨询内容。", "speaker": "A" }
  ],
  "relationship": [
    { "0-0": "root", "0-1": "接收词" }
  ]
}

The dataset includes dependency relation labels like root, sasubi, obj, pred, and more. Detailed explanations can be found in the CODT guidelines.
